* **项目简介**：用来输出技术文章，用来实践学到的技术  
* **使用到的技术**  
	* Nuxt / uni-app / Vuex  / Axios  / SCSS /
	*  Koa / MongoDB / PM2 /Nginx
* **项目难点** 
	* 博客站点的技术架构并不是很复杂，这个站点我主要的精力花在了**移动端适配**和**性能优化**上

	* 使用 vw + rem + 媒体查询 的方案做了移动端屏幕的适配，可以很好的适配各种大小的屏幕。
	* 尝试使用 uni-app 开发了全平台的博客小程序
